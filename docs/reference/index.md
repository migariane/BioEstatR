# Package index

## All functions

- [`freq()`](https://migariane.github.io/BioEstatR/reference/freq.md) :
  Obtencion de la tabla de frecuencias absolutas y relativas de las
  categorias de una variable
- [`get_msg()`](https://migariane.github.io/BioEstatR/reference/get_msg.md)
  : Get multilingual message
- [`grps()`](https://migariane.github.io/BioEstatR/reference/grps.md) :
  Tabla de medias y desviaciones por niveles de un factor
- [`grpsggp()`](https://migariane.github.io/BioEstatR/reference/grpsggp.md)
  : Diagramas por grupos
- [`icl()`](https://migariane.github.io/BioEstatR/reference/icl.md) :
  Estimacion del parametro de la distribucion de Poisson
- [`icm()`](https://migariane.github.io/BioEstatR/reference/icm.md) :
  Estimacion de la media de una variable aleatoria con distribucion
  normal
- [`icp()`](https://migariane.github.io/BioEstatR/reference/icp.md) :
  Estimacion de una proporcion binomial por los metodos: Wilson (con
  cpc), Wald (con cpc) y Wald ajustado
- [`nl()`](https://migariane.github.io/BioEstatR/reference/nl.md) :
  Tamano de muestra para estimar el parametro de la distribucion de
  Poisson
- [`nm()`](https://migariane.github.io/BioEstatR/reference/nm.md) :
  Tamano de muestra para estimar la media de una variable aleatoria con
  distribucion normal con una precision determinada
- [`np()`](https://migariane.github.io/BioEstatR/reference/np.md) :
  Tamano de muestra para estimar una proporcion binomial
- [`osteo`](https://migariane.github.io/BioEstatR/reference/osteo.md) :
  Datos sobre pacientes diabeticos
- [`rlm()`](https://migariane.github.io/BioEstatR/reference/rlm.md) :
  Regresion lineal multiple
- [`rlogitm()`](https://migariane.github.io/BioEstatR/reference/rlogitm.md)
  : Regresion logistica multiple
- [`rlogits()`](https://migariane.github.io/BioEstatR/reference/rlogits.md)
  : Regresion logistica simple
- [`rls()`](https://migariane.github.io/BioEstatR/reference/rls.md) :
  Regresion lineal simple
- [`tabla2x2()`](https://migariane.github.io/BioEstatR/reference/tabla2x2.md)
  : Analisis de una tabla de contingencia 2x2
- [`tablarxc()`](https://migariane.github.io/BioEstatR/reference/tablarxc.md)
  : Analisis de una tabla de contingencia RxC
- [`testf()`](https://migariane.github.io/BioEstatR/reference/testf.md)
  : Test de Fisher (F-test) para comparar dos varianzas
- [`testmcnemar()`](https://migariane.github.io/BioEstatR/reference/testmcnemar.md)
  : test de homogeneidad de dos proporciones apareadas (McNemar)
- [`testnormal()`](https://migariane.github.io/BioEstatR/reference/testnormal.md)
  : Estudio de la normalidad de una variable
- [`testp()`](https://migariane.github.io/BioEstatR/reference/testp.md)
  : Test con proporciones
- [`testt()`](https://migariane.github.io/BioEstatR/reference/testt.md)
  : Test de student con una y dos muestras (independientes o apareadas)
- [`testwx()`](https://migariane.github.io/BioEstatR/reference/testwx.md)
  : Test no parametricos con dos muestras independientes
  (Wilcoxon/Mann-Whitney) y apareadas (Wilcoxon)
