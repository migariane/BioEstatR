# Contribuyendo a BioEstatR

¡Gracias por tu interés en contribuir a BioEstatR!

## Cómo contribuir

1.  **Reportar problemas:** Si encuentras un error o tienes una
    sugerencia, por favor abre un *Issue* en el repositorio de GitHub.
2.  **Enviar mejoras (Pull Requests):**
    - Haz un *fork* del repositorio.
    - Crea una rama para tu mejora (`git checkout -b mi-mejora`).
    - Realiza tus cambios y asegúrate de que el paquete pasa
      [`devtools::check()`](https://devtools.r-lib.org/reference/check.html).
    - Envía un *Pull Request* explicando tus cambios.
